# Buttons

These settings let you control the appearance of buttons across your store. You can choose the button font, toggle uppercase text, and adjust letter spacing to match your brand’s tone—whether it’s bold and modern or clean and minimal.

| Setting               | Description                                                                 |
|------------------------|-----------------------------------------------------------------------------|
| **Button font**         | Choose between your theme’s body or heading font. |
| **Uppercase button text**         | Show all buttons as uppercase. |
| **Letter spacing**         | Control the space between individual characters. Tweaking letter spacing can improve legibility within your buttons. |
